# Telepulse

**Telepulse** is a complete Telegram automation tool designed for real-time message forwarding and full history cloning. It's built to be 100% free, unlimited, and easy to use for any Telegram administrator.

## 🚀 Key Features

- **Real-Time Forwarding (24/7):** Monitor multiple channels or groups and distribute messages automatically as they are posted.
- **Full History Cloning:** Copy all old messages and media from one source to another. Perfect for backups or migrations.
- **Smart Text & Link Substitution:** Automatically replace specific text, entire messages, or just links to customize your content.
- **Premium Emoji Preservation:** Full compatibility with exclusive Telegram premium emojis, formatting, and media albums.
- **Versatile Support:** Works seamlessly with both channels and groups, whether they are public or private.
- **100% Free & Unlimited:** No hidden costs, no task limits, and no channel limits.

## 🛠️ How It Works

1. **Access the Bot:** Open [@encaminhadorabot](https://t.me/encaminhadorabot) on Telegram.
2. **Connect Your Account:** Follow the instructions to link your Telegram account securely.
3. **Configure Tasks:** Set up your forwarding or cloning tasks through the bot or the dashboard.
4. **Start Automating:** Hit "Start" and let Telepulse handle the rest.

## 💻 Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Motion (Framer Motion), Lucide React.
- **Backend:** Node.js, Express, Telegram API (via `telegram` library).
- **Deployment:** Vercel / Cloud Run.
